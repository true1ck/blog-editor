# Blog Editor Frontend

React + Vite frontend for the blog editor application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
VITE_API_URL=http://localhost:5000
```

3. Start development server:
```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- TipTap rich text editor
- Auto-save functionality
- Image upload to S3
- JWT authentication
- Responsive design
